# Personal Portfolio & Research Repository

This is the source code for my professional website and research archive, hosted at [vedranavidulin.com](https://vedranavidulin.com).

The site serves as a digital portfolio showcasing my professional expertise and a comprehensive archive of my academic and technical publications.


## Tech Stack

*   **React 18** (Vite-powered)
*   **TypeScript** for type-safe and maintainable code
*   **Material UI (MUI)** for consistent and accessible design components
*   **GitHub Actions** for automated CI/CD and deployment to GitHub Pages


## Development Setup

Use the following commands for local development:

```bash
# Install dependencies
npm install

# Start development server with hot-reload (localhost:5173)
npm run dev

# Build the project for production
npm run build

# Preview the production build locally
npm run preview
```


## Deployment Workflow

The project is automatically deployed via GitHub Actions. Every push to the main branch triggers a build.
